Load ID Numbering

  • 1xxx - Loads on Master 1
  • 2xxx - Loads on Master 2
  • Examples: 1101 = Master 1, Load 101 | 2225 = Master 2, Load 225

Stations & Buttons Format

Button configurations are automatically extracted from your Vantage system: 

{
  "station_23": {

    "station": 23,
    "master": 1,
    "buttons": {
      "button_1": {
        "name": "Island Pendant",
        "button": 1,
        "event_type": "DIM",
        "loads": [1201]

      "button_5": {
        "name": "All On",
        "button": 5,
        "event_type": "PRESET_ON",
        "loads": [1201, 1202, 1203, 1204]
      }
    }
  }
}

Finding Your Load IDs

  1. Open Vantage Design Center
  2. Go to Loads ‚Üí View load assignments
  3. Note the load numbers (e.g., Load 101, Load 225)
  4. Prefix with master number: Master 1 Load 101 = 1101

Station Numbers

Station numbers match your Vantage keypad addresses:

  • V01-V73 - Keypad stations throughout your home
  • Station 8 (Global) - System-wide controls
  • Check your Vantage config file for exact station numbers

Automatic Extraction

The easiest way to configure is to extract from your Vantage backup:

  1. Export your Vantage configuration from Design Center
  2. Save the file to Info/Home Prado Ver.txt
  3. Run python scripts\extract_buttons.py
  4. All stations, buttons, and loads will be automatically configured!
